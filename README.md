# Node.js-Express-With-MSSQL
This is a demo for node.js express (jade) with mssql  

## Tech Stack
- [X] express
- [X] jade(pug)
- [X] mssql  

## Database
+ express-demo.bak
+ change config/db.js
  
## Install
```bash
git clone https://github.com/explooosion/Node.js-Express-With-MSSQL.git
```  
```bash
npm install
```  
  
+ run with nodemon
```bash
npm install nodemon -g
npm run dev
```  
+ run with pm2
```bash
npm install pm2 -g
npm start
npm stop
```
